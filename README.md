# Cancer Prediction Using Machine Learning

This project focuses on predicting cancer diagnosis using machine learning techniques.
The dataset includes demographic, lifestyle, and biological risk factors.

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)


## Tools & Libraries
- R / RStudio
- tidyverse
- caret
- ggplot2
- pROC

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC Curve and AUC

## Dataset
Publicly available Kaggle dataset:
https://www.kaggle.com/datasets/rabieelkharoua/cancer-prediction-dataset
